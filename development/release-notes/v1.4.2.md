# Highlights

- Stable hosted releases now publish through the Cloudflare custom download domain at `downloads.veldmuislinux.org`.
- The website reads the stable GitHub release data and points the download button at the hosted ISO path automatically.
- Downloaded ISO files now use the release version in the filename while the R2 bucket still keeps only the latest hosted artifacts.

# Notes

- This release validates the full hosted release path after the custom download domain move.
- R2 still stores only the latest ISO, checksum, and manifest objects under the configured prefix before each new release upload.
