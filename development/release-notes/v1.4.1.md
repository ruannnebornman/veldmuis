# Highlights

- Stable hosted releases now build, sign, and publish from a single tag-triggered GitHub workflow.
- The website download path and the R2 bucket now stay aligned on `latest.iso`, `latest.iso.sha256`, and `latest.manifest.txt`.
- Signed package publishing to `packages.veldmuislinux.org` runs as part of the same release flow.

# Notes

- This is the first stable release on the hosted pipeline, intended to validate the long-term release path end to end.
- The website should pick up this stable release from GitHub automatically and point the download button at the hosted ISO path without another website deploy.
